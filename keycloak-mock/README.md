# Keycloak Mock Setup

Local Keycloak setup for SSO testing.

## Quick Start

**Install Keycloak**:
   ```bash
   ./install.sh
   ```

Once keycloak is installed, running `make dev` from the root directory will start keycloak and all the other development servers.
